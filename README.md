# Diseño de Interfaces

# Sistema de Gestión de Equipos de Laboratorio

## Taller de Programación

Este proyecto es un ejemplo académico desarrollado en la asignatura **Taller de Programación** para demostrar cómo construir progresivamente un sistema de gestión aplicando los conceptos vistos en clase.

El sistema permite registrar equipos de laboratorio, consultar su disponibilidad y gestionar préstamos.

---

# Objetivo

Desarrollar un sistema de consola que permita administrar el registro, préstamo y consulta de equipos de laboratorio, aplicando principios de programación estructurada, manejo de excepciones y trabajo colaborativo mediante GitHub.

---

# Alcance

El sistema permitirá:

- Registrar equipos
- Consultar equipos disponibles
- Registrar préstamos
- Registrar devoluciones

Este sistema se implementará en consola y almacenará los datos temporalmente en memoria.

---

# Requerimientos funcionales

El sistema debe permitir:

- Registrar nuevos equipos
- Consultar lista de equipos
- Registrar préstamos
- Registrar devoluciones
- Mostrar disponibilidad de equipos

---

# Requerimientos no funcionales

- El sistema debe ejecutarse en consola
- El sistema debe manejar errores de entrada del usuario
- El código debe estar organizado en módulos
- El sistema debe mantenerse en un repositorio GitHub
